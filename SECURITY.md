# Security Policy

## Supported Versions

The latest release is the supported version of Buddy.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability.

Report security concerns through GitHub's private vulnerability reporting for this repository, or contact the maintainer directly if private reporting is unavailable.

Buddy is designed to run locally inside VS Code. Security-sensitive changes should preserve that model: no telemetry, no source-code upload, and no required external service.
